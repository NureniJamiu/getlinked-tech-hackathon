## Local Installation

```bash
# Clone the repository
git clone https://github.com/NureniJamiu/getlinked-tech-hackathon.git

# Change to the project directory
cd getlinked-tech-hackathon

# Install dependencies
npm install

# Run on development server
npm run dev
```
